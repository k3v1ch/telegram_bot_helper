# Telegram Digest Bot

Юзербот, который ежедневно собирает сообщения из указанного топика Telegram-группы, отправляет их в Gemini для анализа и публикует структурированный дайджест в ваш личный чат/топик.

## Что делает

- Читает сообщения за последние 24 часа из указанного топика группы
- Отправляет лог в Gemini API для формирования сводки
- Публикует дайджест с группировкой по важности в целевой топик
- Работает по расписанию (по умолчанию — 09:00 MSK)

## Требования

1. **Telegram API credentials** — получить на [my.telegram.org](https://my.telegram.org):
   - `API_ID` и `API_HASH`

2. **Gemini API key** — получить на [aistudio.google.com](https://aistudio.google.com/apikey) (бесплатный тариф)

3. **Docker** и **docker-compose**

## Формат SOURCE и DEST

Оба параметра задаются в формате `chat_id:topic_id`, например:

```
SOURCE=-1003332852289:155
DEST=-1003332852289:220
```

### Как узнать chat_id

- Перешлите сообщение из группы боту [@userinfobot](https://t.me/userinfobot) — он покажет числовой ID (например `-1001234567890`)

### Как узнать topic_id

1. Откройте группу в [Telegram Web](https://web.telegram.org)
2. Перейдите в нужный топик
3. ID топика — число после `/topics/` в URL, либо `message_id` первого сообщения в топике

## Установка и запуск

### 1. Клонировать и настроить

```bash
git clone <repo-url>
cd telegram-digest
cp .env.example .env
# Заполнить .env своими данными
```

### 2. Первый запуск (авторизация)

При первом запуске Telethon запросит номер телефона и код подтверждения. Запускайте интерактивно:

```bash
docker compose run digest
```

После успешной авторизации сессия сохранится в `./sessions/`.

### 3. Запуск в фоне

```bash
docker compose up -d
```

## Ручной запуск дайджеста

```bash
docker compose run digest --now
```

## Логи

Логи пишутся в `./logs/digest.log` с ротацией (макс. 5 МБ, 3 файла).

Просмотр в реальном времени:

```bash
docker compose logs -f digest
```

## Структура дайджеста

Бот формирует дайджест с разделами:

- 🔴 **Важное** — критические объявления, breaking changes
- 🟡 **Обновления** — изменения конфигов, новые фичи
- 🔵 **Обсуждения** — полезные технические дискуссии
- ℹ️ **Прочее** — мелочи, заслуживающие внимания

Пустые секции не отображаются. Если чат был тихим — отправляется короткое уведомление.
